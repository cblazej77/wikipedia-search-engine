# wikipedia-search-engine

Wymagania:
- python w wersji 3.12.3

Aby uruchomić program, należy wykonać kroki:
1. Pobierz wszystkie potrzebne biblioteki:
	pip install -r requirements.txt
2. Przejdź do katalogu programu "wikipedia-search-engine/"
3. Uruchom plik wikipedia_search_engine.py:
	python wikipedia_search_engine.py
4. Wybierz 1 z opcji:
    [S] -> Start programu
    [T] -> Uruchom testy
    [x] -> Zakończ działanie
	
Pliki, z których korzysta wyszukiwarka znajdują się w katalogu /data.
