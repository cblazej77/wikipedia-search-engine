About the interwiki links in this page [ edytuj kod ] This page is very special. When any article contains the pattern ISBN 0-12-345678-9 , a link is created to Specjalna:Książki (Special:Booksources), which takes its contents from this page Wikipedia:ISBN , and substitues "MAGICNUMBER" with the specified ISBN. The second level of this magic trick is to use the same MAGICNUMBER in the interwiki links, like this: [[ar:Special:Booksources/MAGICNUMBER]] [[cs:Special:Booksources/MAGICNUMBER]] ... Thus, when a user clicks the ISBN (try it!), in this case for an American book (ISBN starting with 0-12-), and perhaps don't find it in the specified (Polish) booksources, they can click the interwiki link to arrive at the English (or Chinese, or Icelandic) Wikipedia's Special:Booksources page, using the same ISBN. -- LA2 17:26, 17 paź 2007 (CEST) [ odpowiedz ] Linkowisko księgarni komercyjnych [ edytuj kod ] Wg mnie w zgodzie z naszymi zasadami mogą tu być tylko biblioteki.
